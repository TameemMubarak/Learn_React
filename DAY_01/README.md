# DAY_01 — React Learning

Brief README for this folder used to practice and learn React.

## Purpose
- Workspace for Day 1 React exercises and experiments.
- Small, disposable project space to try components, state, and basic hooks.

## Suggested setup
1. Ensure Node.js and npm (or yarn) are installed.
2. Initialize a project if none exists:
    - Using Create React App: `npx create-react-app .`
    - Or with Vite: `npm create vite@latest . --template react`
3. Install dependencies: `npm install`
4. Start dev server: `npm start` or `npm run dev` (Vite)

## Contents
- src/ — source code (components, styles, tests)
- public/ — static files (if present)
- README.md — this file
- package.json — project scripts and deps (if initialized)

## Tips
- Keep commits small and frequent.
- Experiment with small components and use the React docs for reference: https://reactjs.org/docs/getting-started.html

Simple and focused — use this folder to learn by building.
